Wireshark should run in monitor mode
or wireless interface should be configured in monitor mode so that it can monitor the wireless packets

In wireshark preferences select option to decrypt 802.11 traffic and add keys something like

WPA PSK  = Add plain text wifi pre shared key
WPA PWD = Plaintextsharekey:SSIDinplaintext
WPA PWD = Plaintextsharekey


Then start capturing the packets, you will be able to see TCP/IP packets
