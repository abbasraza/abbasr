I was trying to automate the reloading of my email web page on firefox and then i found "Firefox ReloadEvery Addon"

Reloads web pages every so many seconds or minutes. The function is accessible via the context menu (menu you get when you right click on a web page) or via tab context menu (just in Firefox 4.0)

But unfortunately it is not applicable to mail-provider-websites, communities, chats etc., because it will also reload the web page while composing a new message. So that message will be lost/deleted when the website reloads :-(

Anyway this add-on can be useful  for some other purpose.
