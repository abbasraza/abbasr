  1. Select some music file you want to use. Like main.mp3. Play it in some player and find out the total duration of this track. Like in my case it was 4:20 minutes. Which means 260 Seconds.
  2. Now grab some photos/images you want to add to this video. I had 37 images which means that i need to play an image for 7 Secs. so every image will be displayed for 7 secs totaling to 259 secs (37 * 7  = 259)
  3. Re-size all your images to 640x480 resolution.

    $ convert -resize 640x480 input.png output.png

    ‘convert’ is part of the ImageMagick suite. If you don’t have it already, you can install that with:

    $ sudo apt-get install imagemagick
  4. Now it’s time to create the slideshow. For that we need dvd-slideshow.
    $ sudo apt-get install dvd-slideshow
  5. dvd-slideshow runs from a recipe or script . Lets create this
    abbas@dell:dtemp$ cat rec.txt    main.mp3:1
    title.jpg:10
    fadein:2

    0/s9-14.jpg:7
    crossfade:2
    0/s9-15.jpg:7
    crossfade:2

    1/s9-0.jpg:7
    crossfade:2
    1/s9-2.jpg:7
    crossfade:2
    1/s9-3.jpg:7
    crossfade:2
    1/s9-4.jpg:7
    crossfade:2
    1/s9-6.jpg:7
    crossfade:2
    ..........
    ..........
    5/s9-25.jpg:7
    fadeout:2
    5/s9-26.jpg:7

    title.jpg will be shown for 10 secs and after that a crossfade of 2 secs and so on...
  6. Run dvd-slideshow as follows:

    $ dvd-slideshow rec.txt
  7. you will get a .vob file after this . You can play this video file and see the results.
  8. This .vob file will be having large size. So to convert it to avi

    $ sudo apt-get install mencoder lame
  9. Now create a script to convert .vob to avi

    abbas@dell:dtemp$ cat avi.sh
    mencoder $1 \
    -ovc lavc -lavcopts
    codec=mpeg4:vbitrate=1000:vhq:keyint=250:threads=2:vpass=1 \
    -oac mp3lame -lameopts cbr:br=128 \
    -ffourcc XVID \
    -vf scale=640:-2,crop=640:480,expand=640:480 \
    -af resample=44100:0:0 \
    -o $2

    mencoder $1 \
    -ovc lavc -lavcopts
    vcodec=mpeg4:vbitrate=1000:vhq:keyint=250:threads=2:vpass=2 \
    -oac mp3lame -lameopts cbr:br=128 \
    -ffourcc XVID \
    -vf scale=640:-2,crop=640:480,expand=640:480 \
    -af resample=44100:0:0 \
    -o $2
  10. Now run this script and you will get your final .avi video file.

    $ ./avi.sh rec.vob rec.avi
  11. And finally you can upload this video file to youtube and share it :)
