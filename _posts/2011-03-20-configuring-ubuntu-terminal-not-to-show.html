If you want your terminal not to show complete directory path on command prompt then

  1. Edit /home/user/.bashrc and replace 'w' in PS1 variable with 'W' like this
    PS1='${debian_chroot:+($debian_chroot)}\u@\h:\W\$ '
  2. Run following command to apply changes to your teminal "source /home/user/.bashrc"
