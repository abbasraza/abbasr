Finally on
A): Add 'synergys -f --config /etc/synergy.conf ' to startup applications.
B): Add ' synergyc 137.202.157.57' to startup applications.

Misc:
In case you get error
abbas@abbas:linux-kernel$ synergys -f --config /etc/synergy.conf
synergys: /usr/lib/libstdc++.so.6: version `GLIBCXX_3.4.14' not found (required by synergys)

Remedy:
abbas@abbas:linux-kernel$ sudo apt-get remove synergy
abbas@abbas:linux-kernel$ sudo apt-get install synergy
