I have vim7.2 and here are simple steps to start using cscope with vim

  1. Make sure you have cscope install on your Linux machine, if not the install it.

  2. Download the cscope_maps.vim file, and arrange for it to be read by Vim at startup time. If you are using Vim 6.x, stick the file in your $HOME/.vim/plugin directory .

    (If you are using Vim 5.x, you can either cut and paste the entire contents of the cscope_maps file into your $HOME/.vimrc file, or stick a "source cscope_maps.vim" line into your .vimrc file)

  3. Goto your source directory and run commands there

      # find . -name "*" > cscope.files


      # cscope -b

    It will cscope database and generate cscope.out file

  4. export CSCOPE_DB path

    # CSCOPE_DB=$PWD/cscope.out
    # export CSCOPE_DB

  5. Open any source file with vim and apply command in vim . e.g to find list of functions which are calling the function "packet_setup" give following command

    cscope find c packet_setup

    OR

    cs f c packet_setup

    You can use options give below as required...

    c: Find functions calling this function
    d: Find functions called by this function
    e: Find this egrep pattern
    f: Find this file
    g: Find this definition
    i: Find files #including this file
    s: Find this C symbol
    t: Find assignments to

** In case of duplicate database error modify cscope_maps.vim and replace


  if filereadable("cscope.out")                             
        cs add cscope.out 


with


if exists("./cscope.out")
        cs add cscope.out

** You can modify key mappings in cscope_maps.vim  according to your choice
