The file cert8.db in your profile folder may have become corrupted. Delete this file while Firefox is closed.

abbas@dell-ubuntu:~$ find .mozilla/ -name cert8.db
.mozilla/firefox/rhes4mfk.default/cert8.db
abbas@dell-ubuntu:~$ rm -rf .mozilla/firefox/rhes4mfk.default/cert8.db
 
Restart Firefox.
