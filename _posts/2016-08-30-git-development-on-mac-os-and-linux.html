Linux git has different endofline scheme, so when you checkout a git repo on MAC OS,
its different if you access the same repo (via VM shared folder) on linux. To sync both linux and MACOS on endofline character do this on MAC:

  cd git-repo;
  git config --global core.autocrlf input
  git rm --cached -r .
  git reset --hard

Now you the shared repo between MACOS and linux will be having same line endings.

have fun!
