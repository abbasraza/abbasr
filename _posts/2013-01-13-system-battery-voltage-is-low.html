Nonvolatile BIOS Memory1)The BIOS Setup program allows the user to view and modify various hardware configuration settings, as shown above.  However, these settings are not stored in the ROM BIOS chip itself.
Historically, the configuration settings could not be stored in the ROM BIOS chip, because they needed to be updated dynamically (ie, when the PC was booting, when a new device was added, when the user wanted to change a password, etc.).  This would have been impossible with EPROM or EEPROM chips, which needed to be removed and inserted into a programmer device in order to be modified.
Instead, the settings were stored in another type of memory chip known as CMOS (complementary metal oxide semiconductor).  CMOS was a similar technology to SDRAM, except that the circuitry required much less power - in fact, the chip could be powered by a small lithium "button" battery, and therefore, could retain it's settings even when the PC was not powered on.  As a result, the chip was called the "CMOS" chip, even though the CMOS acronym reflects the IC technology used to manufacture the chip, and not the contents or function of the chip.
Today, the BIOS settings are normally stored on a flash ROM chip somewhere on the motherboard, separate from the BIOS chip.  The correct term for this chip is now "nonvolatile BIOS memory"; however, the term CMOS is still commonly used.  The small lithium button battery is now used only to power another small chip, which controls the real-time clock on the PC.

2)The CMOS Battery supplies power to the system board to keep the time and the BIOS settings. If you pull the battery and then let the board discharge (up to an hour, depending on the board) the date will default to the earliest time in the BIOS version, The Bios will loose its custom settings and any BIOS password will be deleted.

Keeps the System Date and TimeThe CMOS battery stores the system time and settings that most be loaded when you turn the system on. Older computers reset to a date of 1-1-1980. Computers running Windows XP or later will set the system date to the date of the last saved system restore point.
Allows System Settings to be called upA few additional settings are stored by the system. The CMOS battery allows these settings to be loaded into system memory when the computer boots.
Why does it look like a watch battery?The CMOS battery is a standard-size watch battery in most desktop systems. The user can buy a replacement at any place that sells watches.
How long does this battery last?CMOS batteries can last for 5 years. If a computer's CMOS battery loses a charge, you will have to manually set the system date and time.
What does CMOS stand for?CMOS is the acronym for Complementary Metal Oxide Semiconductor.
