Capturing audio using alsa utils and how to specify the capture and playback device:

Device, sub device numbers can be retrieved from /proc

root@crossville:~# ls /proc/asound/card0/
codec#0  id       pcm0c    pcm0p    pcm1c    pcm1p    pcm2c
root@crossville:~# cat /proc/asound/card0/pcm0c/info 
card: 0
device: 0
subdevice: 0
stream: CAPTURE
id: ALC888 Analog
name: ALC888 Analog
subname: subdevice #0
class: 0
subclass: 0
subdevices_count: 1
subdevices_avail: 1
root@crossville:~# 
root@crossville:~# cat /proc/asound/card0/pcm2c/info
card: 0
device: 2
subdevice: 0
stream: CAPTURE
id: ALC888 Analog
name: ALC888 Analog
subname: subdevice #0
class: 0
subclass: 0
subdevices_count: 1
subdevices_avail: 1
root@crossville:~# 


arecord -f cd -D hw:0,2 | aplay -f cd -D hw:0,0

I used device #2 of card0 which is a capturing device for recording and device #0 of card0 which is both playback & capturing device as playback.
