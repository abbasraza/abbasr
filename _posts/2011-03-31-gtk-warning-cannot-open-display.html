After new ubuntu installation on my system i was trying to open wiresharkon terminal but was getting this error

root@localhost:~# sudo wireshark

(wireshark:21267): Gtk-WARNING **: cannot open display: 


Actually it was happening with any gui application .. then i found solution for it.

run command

root@localhost:~# export DISPLAY=:0.0

Now you can run gui successfully :)
