guest additions don't come by default. you need them to use facilities like copy paste between host and guest + sharing folders etc

start VM, in menu above you will go Devices--> insert guest additions CD image. images comes with virtual machine?? no need to install/download

it will ask to run. run it

if face issues like below, install headers before and redo. if again issues, update virtualbox software. goto menu checkforupdates. download and re-reun.

issues will go away.



$ sudo apt install linux-headers-$(uname -r) build-essential dkms
