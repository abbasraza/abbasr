In 2010, and I wouldn't be surprised if still issue in current version, OneNote corrupts source code pasted/copied.  For example, I was using OneNote to keep snippets of cshtml code, but when copying that code from OneNote, some characters are not preserved in their original pasted form.  Quotes become smart(dumb) quotes.  Space become a non-printable character that appears to be a space, but is not, and as such it breaks the code.  The space becomes a no-break-space U+00A0.  This breaks any HTML or CSHTML that is pasted, because the spaces inside element tags between attributes, or within attributes, will be encoded and no longer treated as whitespace.  This attributes do not work.  This is all because OneNote doesn't preserve the space character properly.
