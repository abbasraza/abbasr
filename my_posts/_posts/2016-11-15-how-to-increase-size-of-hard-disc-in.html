On Mac OS:

resizing the ubuntu virtual machine disk from 8GB to 16GB
1) shutdown ubuntu VM (shutdown it from ubuntu, not save/suspend)
2) if your vm is using vmdk disk image, then clone it into vdi image first

vboxmanage clonehd ~/VirtualBox\ VMs/Ubuntu\ 15.04/Ubuntu\ 15.04-disk1_1.vmdk ~/VirtualBox\ VMs/Ubuntu\ 15.04/Ubuntu\ 15.04-disk1_1.vdi --format vdi

3) resize to 16GB
VBoxManage modifyhd ~/VirtualBox\ VMs/Ubuntu\ 15.04/Ubuntu\ 15.04-disk1_1.vdi --resize 16384

4) Goto this virtual machine settings --> storage --> add new disk to controller --> existing disc image vdi from 3) step path, make it hot pluggable. uncheck old disk hot plug option.

5) download gparted-live iso like this gparted-live-0.27.0-1-i686.iso

6) Add this iso as optical drive in settings --> controller IDE --> iso image

7) now start VM

8) you will run into gparted

9) delete swap and extended partitions

10) now extend root partition, leave 1/2 GB for extended/swap
Need to select device /dev/sdb instead of /dev/sda


12) create new extended and swap partitiions on 1/2GB

13) save changes

14) exit gparted

15) remove iso from optical drive settings of VM

16) run VM
