  1.  to see number of threads of PID: 29138
    top -p 29138
  2. hit shift h
  3. to see on which processor the thread is running/sleeping
  4. hit f
  5. then using up/down key to go to a parameter you want to appear in top
  6. let's say "  P       = Last Used Cpu (SMP)"
  7. hit spacebar to select that parameter
  8. now hit right arrow key to select that parameter
  9. then hit up to move this parameter to up in the list
  10. the hit escape button to return to top display
  11. now you will be able to see the new parameter in display
