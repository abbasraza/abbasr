check if port 1291 is

  • OPEN:  the firewall allows access to this port and also a software is listening on this port
  • CLOSED: the firewall allows access to this port and but no software is listening on this port
  • FILTERED: the firewall doesn't allows access to this port
nmap -p 1292 35.157.191.242
