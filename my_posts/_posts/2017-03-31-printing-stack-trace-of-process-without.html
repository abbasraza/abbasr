Use gdb batch feature


ubuntu@bifrost:~/abbas$ cat stack.sh
#!/bin/bash
pid=$1
EXE=`readlink -f /proc/$pid/exe`
DIR=`dirname $EXE`
cd $DIR
gdb $EXE --batch -ex "attach $pid" -ex "thread apply all bt"

ubuntu@bifrost:~/abbas$ sudo ./stack.sh PID

ubuntu@bifrost:~/abbas$ sudo ./stack.sh 29138

to change print settings:
https://sourceware.org/gdb/onlinedocs/gdb/Print-Settings.html

to print long string add following option in above script:
-ex "set print elements unlimited"
