'EXT3-fs: error loading journal' error is really confusing one.Simple solution to this error is checking file system integrity.

Run command

# e2fsck /dev/sdax 

or 

# fsck /dev/sdax

where sdax is your Linux partition.It will then ask to replace or fix corrupted blocks and etc.Accept everything by typing 'Y'.

It worked for me.
