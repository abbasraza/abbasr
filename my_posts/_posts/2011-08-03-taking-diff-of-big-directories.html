Taking diff of big directories


I was using 'meld' to see the diff of linux kernel directories. But i was only able to see the diff and i could not find any option in meld to save the diff of directories in 'patch' format.


Then i used standard diff command :)




diff -Naur /home/abbas/work/linux-2.6.32 /home/abbas/work/kernels/linux-2.6.32_lttng/ -X pats > k


Where 'pats' file contains patterns which we want to ignore while taking diff. -X option is used to exclude all the files which match any pattern in 'pats' file.


Here is my 'pats' file


################################
*.ko
*.jpg
*.gif
*.png
*.wav
*.mp3
*.ogg
*.xcf
*.xpm
*.pyc
*.a
*.obj
*.o
*.so
*.la
*.lib
*.dll
*.bzr
*.hg
*.git
*.svn
*.cdv
*.cmd
*.gz
*.out
*.files
*.in
*.po
*.maps
*vmlinux*
uImage
################################




so *.ko, *.o etc files will be ignored while diff is taken.
