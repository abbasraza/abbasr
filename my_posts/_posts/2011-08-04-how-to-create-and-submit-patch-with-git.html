The preferred way of creating a patch is to create it against current git. Follow procedure




1) git clone (clone repo on which you will be working on)


2) Make changes, hack hack hack (make changes in your cloned git branch)


3) Use git gui to view changes you made


$ sudo apt-get install git-gui
$ git gui 



4) Stage the changes you want to commit


5) Write a commit message in commit window


6) sign-off changes


7) Commit changes


8) Quit gui


9) get name of your branch


$ git branch 
* master


10) Create patch with your changes


$ git format-patch origin/master


This will create patch for you with sign-off info, commit message which can be submitted by email.
