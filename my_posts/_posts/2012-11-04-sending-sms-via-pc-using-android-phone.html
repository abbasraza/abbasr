Install on Phone:

*DeskSMS

https://play.google.com/store/apps/details?id=com.koushikdutta.desktopsms

*Connect to Internet on phone and enable Google account.

*Allow DeskSMS to use google account on your phone.

Install on PC:

*Google Chrome plugin of DeskSMS
  
Chrome Extension - http://bit.ly/mPlZuy
FireFox Extension - http://bit.ly/owfySU
DeskSMS Web Interface - https://desksms.appspot.com
